# Caso de Sucesso: HubSpot escala a geração de imagens em 150% com inteligência artificial de estabilidade no Amazon Bedrock

## EMPRESA
A HubSpot é uma plataforma global de CRM voltada para marketing, vendas e atendimento ao cliente. Ela permite que empresas centralizem todas as interações com seus clientes em um único sistema. Por operar no modelo SaaS, tudo funciona na nuvem, com milhares de usuários acessando ao mesmo tempo. Isso exige uma base tecnológica capaz de lidar com grandes volumes de dados e acessos simultâneos sem perda de desempenho.

## DESAFIO
Com o crescimento acelerado da plataforma, a HubSpot começou a enfrentar dificuldades típicas de escala. O volume de dados aumentou muito rápido, e a infraestrutura existente já não conseguia acompanhar com eficiência. Os principais problemas eram:
- Processar dados em tempo real sem aumentar demais os custos.
- Garantir que o sistema nunca ficasse indisponível.
- Lidar com bancos de dados cada vez maiores.
- Manter a produtividade da equipe de desenvolvimento, sem sobrecarga com problemas de infraestrutura.

## POR QUE USAR A AMAZON?
Para resolver esses pontos, a empresa adotou a Amazon Web Services. A escolha foi bem estratégica: já que a AWS oferece serviços prontos e escaláveis, elimina a necessidade de gerenciar servidores manualmente. Entre os fatores mais importantes:
- Infraestrutura global, reduzindo latência para usuários ao redor do mundo.
- Serviços gerenciados como o Amazon Aurora, que melhora o desempenho dos bancos de dados.
- Uso do Vitess para escalar bancos de dados de forma distribuída.
- Facilidade para automatizar testes e implantações frequentes.

## BENEFÍCIOS

A adoção da AWS trouxe melhorias claras para a plataforma:
- Sistema mais rápido: redução perceptível no tempo de resposta.
- Custos mais controlados: pagamento baseado no uso real.
- Mais velocidade no desenvolvimento: novas funcionalidades são lançadas com mais frequência.
- Maior confiabilidade: menos risco de falhas mesmo com alta demanda.

## ARQUITETURA
A base da arquitetura é distribuída e orientada a microsserviços, o que permite escalar partes específicas do sistema conforme necessário. Alguns dos principais componentes:
- Amazon EC2: executa as aplicações.
- Amazon S3: armazena arquivos e dados.
- Amazon Aurora: gerencia os dados de CRM com alta performance.
- Sistemas de mensageria e streaming: garantem atualização em tempo real.
- Camadas de segurança da AWS: protegem os dados e garantem conformidade.

No geral, essa arquitetura permite que a HubSpot cresça continuamente sem comprometer desempenho, segurança ou experiência do usuário.
